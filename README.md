# 💬 Varta – Chatting Platform

**Varta** is a modern and simple chatting platform designed to connect people with a clean, fast, and user-friendly experience.

---

## 🚀 About the Project

Varta is built with the idea of making communication easy and accessible.
It focuses on simplicity, performance, and a smooth user interface.

✨ Whether you're chatting with friends or staying connected with family, Varta provides a minimal yet powerful experience.

## 🎯 Features

* 💬 Real-time chatting experience
* 🎨 Clean and modern UI design
* ⚡ Fast and responsive performance
* 📱 Mobile-friendly interface
* 🔒 Secure and reliable platform

## 🛠️ Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: Firebase / Web Services
